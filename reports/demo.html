Architecture

classifyStructured output (Pydantic) — routes to simple / tool / missing_info / risky / error
LLM
answerGrounded response generation — uses tool results + approval context

Logic Nodes

LOGIC
intakeNormalise raw query text
LOGIC
toolMock tool with transient error simulation (attempt < 2)
LOGIC
evaluateHeuristic quality gate — drives retry loop
LOGIC
retryIncrement attempt counter, log error
LOGIC
dead_letterEscalate after max retries exceeded
LOGIC
clarify / risky_action / finalizeMissing-info, action prep, audit close
